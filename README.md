# Awamer Network Company Website

A modern corporate website built for **Awamer Network**, a software and technology solutions company.
The project focuses on clean UI structure, scalability, and responsive design using modern front-end technologies.

---

## 🧠 About The Project

This project is a front-end web application developed as part of a **3-month training program** at **Awamer Network**.

The main goal was to build a professional company website that presents:
- Company services
- Technical expertise
- Corporate identity
- Clean and maintainable UI

The project is functional and stable, with ongoing improvements mainly focused on responsive behavior and UI refinements.

---

## 🛠 Tech Stack

- **Nuxt 3**
- **Vue 3 (Composition API)**
- **Pinia** (State Management)
- **Tailwind CSS**
- JavaScript (ES6+)

---

## 📄 Website Pages

- Home
